# Сайт копицентра

Готовый Node.js сайт для копицентра:
- загрузка файлов клиентами
- выбор формата печати от A5 до A0
- выбор цветности, тиража, сторон печати, материала, постобработки
- сохранение заказов на сервере
- админ-панель с просмотром заказов и сменой статуса

## Быстрый запуск

```bash
cp .env.example .env
npm install
npm start
```

После запуска:
- сайт: `http://SERVER_IP:8080`
- админка: `http://SERVER_IP:8080/admin/login`

## Переменные окружения

Смотри `.env.example`

## Данные

Заказы сохраняются в:
- `data/orders.json`

Файлы сохраняются в:
- `uploads/`

## Продакшн

Для постоянной работы можно использовать `pm2` или `systemd`.
